# TIPS POUR PIXELLAB

## Paramètres Recommandés

- **Resolution** : Spécifier dans le prompt (64x64, 32x32, 16x16)
- **Style consistency** : Toujours inclure "pixel art" + "grimdark" + "muted colors"
- **Transparent background** : Crucial pour le layering

## Workflow Suggéré

1. **Générer le corps de base** en premier
2. **Tester le layering** avec 1-2 équipements
3. **Ajuster les prompts** si le style diverge
4. **Batch generate** une fois le style validé

## Si le Style Diverge

Ajouter ces modificateurs :
- "no glow, no magic effects"
- "realistic medieval, historically inspired"
- "dark souls aesthetic" (si trop cartoon)
- "battle brothers style" (référence directe)

## Couleurs à Forcer si Besoin

```
color palette: #3d3d3d (dark grey), #5c4a3d (brown), #8b7355 (tan), #6b6b6b (iron), #4a0000 (dried blood)
```
